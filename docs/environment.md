## Environment variables (Backend)

| Name                 | Description                                                           |
| -------------------- | --------------------------------------------------------------------- |
| PORT                 | Port number on which the server runs (e.g., 6000)                     |
| MONGODB_URI          | Connection string for MongoDB database                                |
| GOOGLE_CLIENT_ID     | Client ID obtained from Google Cloud Console for OAuth                |
| GOOGLE_CLIENT_SECRET | Client Secret from Google Cloud Console for OAuth authentication      |
| REDIRECT_URI         | URL where Google redirects after successful login (callback endpoint) |
| JWT_ACCESS_SECRET    | Secret key used to sign and verify access tokens                      |
| JWT_REFRESH_SECRET   | Secret key used to sign and verify refresh tokens                     |
